What is the WWW?

www = A collection of html hyper links or links that was invented in the 1990's.

Major Concepts of the Web

Information travels from your computer to the internet. From there it sends a http request to the servers that hold the content you are looking for. That content is then returned to you in HTML which is interputed by your web browser in a readible format.

HTML includes text content, markup, references to other documents like images or videos and may also contain links to other pages.

Markup includes tags which follow this format opening tag < name > contents closing tag < /name > This format in its entirity is called an element.

Tags can allow for the linking of websites or even images

What is Whitespace?

Whitespace is the space between multiple lines which HTML generally ignores. There are several ways to combat this depending on your goal.

A line break can be created by using < br >.

You can block your content by using < p > add content here < /p >
